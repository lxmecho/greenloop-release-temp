# 网站使用文档（用户版）

## 1. 文档说明

本手册面向普通使用者，帮助你完成注册登录、物品提交、查看回收进度和积分兑换等操作。

网站地址：`https://134139.xyz/`

![网站访问二维码](assets/images/site-qrcode-poster.png)

网站适用场景：

- 想把损坏或老旧设备投放到固定回收点
- 想预约管理员上门回收电子废弃物

## 2. 登录与注册

### 2.1 注册账号

1. 打开网站首页
2. 进入“登录 / 注册”
3. 切换到“注册账号”
4. 输入手机号，点击“获取验证码”
5. 填写短信验证码、密码、昵称和所属校区
6. 提交后即可自动登录

### 2.2 登录账号

1. 打开“登录 / 注册”页面
2. 输入手机号和密码
3. 点击“登录”

## 3. 提交物品

登录后进入“提交物品”页面，填写以下信息：

- 物品名称
- 类别
- 品牌 / 型号
- 成色 / 状态
- 文字说明
- 实物图片
- 处理方式

图片要求：

- 支持 JPG、PNG、WebP、GIF
- 建议上传清晰正面图
- 文件大小不超过 5MB

## 4. 两种处理方式说明

### 4.1 投放固定回收点

适用情况：

- 设备损坏或老旧
- 可以自行携带到回收点投放

提交时需要补充：

- 校区
- 园区
- 具体点位
- 预计投放时间

审核通过后：

- 记录进入“待投放”
- 系统生成物品编号
- 你会收到站内通知
- 投递并完成核验后获得积分

### 4.2 上门回收

适用情况：

- 有电子废弃物需要预约回收
- 希望由管理员按预约时段上门处理

提交时需要补充：

- 宿舍楼栋
- 楼层（1-13 层）
- 房间号（示例：1105/1209）
- 可预约时间段

审核通过后：

- 记录进入“待上门回收”
- 管理员会按预约时段上门处理
- 取走后需要你在系统中确认“已取走”
- 完成入仓核验后获得积分

## 5. 查看物品处理进度

进入“个人中心”后，你可以看到：

- 我的提交
- 站内通知
- 兑换记录
- 当前积分

常见状态说明：

- 待审核：管理员尚未处理
- 待投放：固定回收点记录已审核通过
- 已投递待回收：你已确认投递，等待统一回收
- 待上门回收：管理员已确认上门回收
- 已取走待入仓：双方已完成取走确认，等待入仓核验
- 已完成：本条流程已结束
- 已驳回：管理员未通过审核

## 6. 积分与兑换

### 6.1 如何获得积分

- 固定回收点记录完成最终核验后获得积分
- 上门回收记录完成最终核验后获得积分

### 6.2 如何兑换商品

1. 进入“积分兑换”
2. 查看商品所需积分和库存
3. 点击“立即兑换”
4. 等待管理员确认发放

兑换规则：

- 提交兑换申请时，积分会先暂扣
- 若管理员驳回，系统会自动退回积分

## 7. 站内通知

平台会通过站内通知告知你：

- 提交物品是否审核通过
- 是否进入固定回收点或上门回收流程
- 是否完成取走确认或入仓核验
- 积分是否到账
- 积分兑换是否通过

建议登录后优先查看“个人中心 -> 站内通知”。

## 8. 常见问题

### 8.1 提交后多久会有结果

一般要等待管理员审核，具体处理时间以管理员审核进度为准。

### 8.2 为什么我的上门回收没有立刻得分

上门回收积分会在物品完成取走确认并通过最终入仓核验后发放。

### 8.3 为什么我看不到联系方式

平台默认保护用户隐私，不会公开显示手机号等敏感信息。

### 8.4 图片上传失败怎么办

请优先检查：

- 图片格式是否支持
- 图片大小是否超过 5MB
- 是否使用了 HEIC/HEIF 图片

## 9. 使用建议

- 文字说明尽量写清功能状态和配件情况
- 可自行携带的设备优先选择“固定回收点”
- 如希望由管理员到宿舍回收，可选择“上门回收”
- 及时关注个人中心通知，避免错过审核结果

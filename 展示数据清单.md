# 展示数据清单

这份清单用于继续完善网站前端展示内容。  
按照你现在的要求，网站本身不展示问卷结果，所以这里只保留和网站页面直接相关的信息。

## 一、学校基础信息

- 学校全称
- 项目正式名称
- 校区名称

## 二、回收点信息

每个回收点建议提供：

- 回收点名称
- 所属校区
- 具体位置
- 开放时间
- 一句话说明

建议数量：

- 2 到 4 个点位最适合首页展示

## 三、案例展示信息

建议至少准备：

- 1 条捐赠案例
- 1 条固定回收点投放案例

每条案例建议包含：

- 物品名称
- 类别
- 状态
- 一句话说明

## 四、如果你后面继续补内容，优先级建议

优先补充：

1. 学校全称
2. 两个校区名称
3. 2 到 4 个回收点信息
4. 1 条捐赠案例
5. 1 条回收案例

## 五、推荐发给我的格式

```text
学校全称：
项目名称：
校区1：
校区2：

回收点信息：
1. 名称 / 校区 / 位置 / 开放时间 / 说明
2. 名称 / 校区 / 位置 / 开放时间 / 说明
3. 名称 / 校区 / 位置 / 开放时间 / 说明

展示案例：
1. 捐赠案例：物品名称 / 类别 / 状态 / 一句话说明
2. 回收案例：物品名称 / 类别 / 状态 / 一句话说明
```

你把这些信息继续补给我之后，我就可以再把首页、案例展示区和回收点展示区进一步精修。
